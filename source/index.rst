Lomiri Documentation
====================

Welcome to the official documentation for **Lomiri**, a convergent shell for
mobile, desktop, and form-factor-independent operating systems. This project
brings together two companion documents:

* The **Human Interface Guidelines (HIG)** — the principles, patterns, and
  conventions that make Lomiri apps feel like Lomiri apps.
* The **Design System** — the tokens, components, and usage guidelines that
  put those principles into practice.
